Event History Information

The Hide Expired checkbox hides any Expired events.

The Misty Timers checkbox will change the Time Left column to Misty Timer. If editing the event with this checked, you should input the time that Misty provides.

The Clear All button will remove all events.

Button Icons:

  • Removes the expired event.
  • Allows you to update or correct an existing event. Available only to Scouters.
  • Allows moderators to perform a Global Delete Action.

Notification Modes

This setting controls how notifications are delivered for tracked events. You can select one or multiple options below:

  • Tooltip ‚Äì Shows a short message near your cursor for a few seconds.
  • Toolbar ‚Äì Displays a message in your RuneScape title bar for 2 minutes.

You can select multiple modes using Ctrl/Cmd or Shift when clicking. To clear your selections, use the Reset notification modes link below the selector.

If no favorite events are set, all events will trigger notifications. Otherwise, only favorited events will.

Note: Notifications will only be sent if the application is running inside Alt1.
